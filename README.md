# 🌐 Mivalis Web

Repositorio central con todos los proyectos y herramientas del ecosistema **Mivalis** — una tienda en línea de accesorios y tecnología.

---

## 📂 Estructura del Repositorio

| Carpeta | Descripción |
|---------|-------------|
| **costume-store** | Frontend de la tienda web (HTML/CSS/JS) con panel de administración y API en PHP para gestión de productos, pedidos y usuarios. |
| **app y base de datos** | Backend principal (Node.js + PostgreSQL) con frontend de tienda, sistema de pagos, Docker Compose y migraciones SQL. Desplegable en Render. |
| **admin panel** | Panel de administración web y chatbot de bodega para gestión interna de productos e inventario. |
| **Automatización N8N** | Automatizaciones con N8N (Docker Compose) — workflows, webhooks, dashboard y scripts de testing para la integración con la tienda. |
| **chataboto-desu** | Reglas de configuración y personalidad para el chatbot de atención al cliente / ventas por WhatsApp. |
| **Telegram Miniapps** | Proyecto de Mini Apps para Telegram enfocado en salud y bienestar. |

## 🛠️ Tecnologías Principales

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, PHP, Python
- **Base de Datos:** PostgreSQL, SQLite
- **Automatización:** N8N, Docker
- **Deploy:** Render, Railway
